Senior Project SP-103

Course: CS 4850

Semester: Spring 2026

This project explains how black-box machine learning models make predictions. Our team trained an XGBoost model on the Adult Income dataset and used SHAP and LIME to show which features influenced the model’s results.

Project Goal

The goal of this project is to improve transparency in machine learning by providing both global and local explanations of model predictions.
